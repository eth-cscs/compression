Detailed Description

template<typename SCALARTYPE, typename SizeType, bool is_iterator1>
class viennacl::tools::sparse_matrix_adapted_iterator< SCALARTYPE, SizeType, is_iterator1 >

A non-const iterator for sparse matrices of type std::vector<std::map<SizeType, SCALARTYPE> >

The iterator behaves like ublas iterators. Attention: Iteration along first columns and then rows via .begin() is untested! Reverse iterators are missing!

Template Parameters:
SCALARTYPEeither float or double
is_iterator1if true, this iterator iterates along increasing row indices, otherwise along increasiong column indices
